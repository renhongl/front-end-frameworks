# 单页应用的基本架构
单页应用是把多种功能集成到一个页面的网页应用，跟传统网页相比，它的切换更流畅， 响应速度迅速。
那么，单页应用就应该具备一些基本功能：
1. 一个html文件；
2. 可以切换路由；
3. 可以与服务器通信；

好的单页应用，还应该具备以下特点：
1. 具备良好的扩展性，易于增加路由页面和功能模块；
2. 组件粒度小，重用性高；
3. 经过优化，渲染时间短；
